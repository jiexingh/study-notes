# web 3D AR 相关知识

## 计算机3D 图形三要素

1. 模型:多边形网格，材质，骨骼动画
2. 光照:环境光，方向光，射灯
3. 镜头:栅格化

VR :virtual Reality 虚拟现实  
AR :Augmented Reality 增强现实 实时融合虚拟和真实世界
webAR: 无需安装应用，浏览器即可运行
web AR三种方式：标记 图形 地理位置
web 3D 引擎:模型 光照 镜头
